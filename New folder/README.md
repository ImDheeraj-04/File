# Hermynee is Love – Part 2 💕

A magical, interactive romantic website created with Next.js, Tailwind CSS, and Framer Motion. This is Part 2 of a beautiful love story, featuring stunning animations, interactive features, and heartfelt messages.

## 🌟 Features

### 1. **Hero Section**
- Full-screen animated background with floating particles
- Typing animation with rotating love messages
- Background music toggle (ready for implementation)
- Glowing title with pulsing heart animation

### 2. **Love Story Timeline**
- Animated timeline of relationship milestones
- Flip animation cards with romantic messages
- Beautiful gradient backgrounds
- Hover effects and smooth transitions

### 3. **Love Letter**
- Interactive envelope that opens to reveal a heartfelt letter
- Typewriter effect for the letter content
- Paper-style design with decorative elements
- Romantic typography and styling

### 4. **Memory Gallery**
- Polaroid-style photo gallery
- Lightbox functionality with navigation
- Hover animations and effects
- Captions for each memory

### 5. **Why I Love You**
- 12+ flip cards with reasons why you love her
- Beautiful gradient designs
- Hover animations and 3D effects
- Personalized messages on the back

### 6. **Love Quiz**
- Interactive quiz about your relationship
- Progress tracking and scoring system
- Fun questions about your love story
- Animated results with special messages

### 7. **Secret Message**
- Password-protected secret message
- Password hint: Your anniversary date (YYYY-MM-DD format)
- Beautiful unlock animation
- Heartfelt message revealed on success

### 8. **Heart Game**
- Catch falling hearts mini-game
- Score tracking and timer
- High score system
- Fun and interactive gameplay

### 9. **Final Section**
- Glowing, animated heart centerpiece
- Beautiful typography and messaging
- Floating particles and animations
- Emotional conclusion to the experience


   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🔧 Customization

### Personalize the Content:

1. **Update names and messages** in the component files
2. **Change the password** in `SecretMessage.tsx` (line 11)
3. **Add your own photos** to the Memory Gallery
4. **Customize quiz questions** in `LoveQuiz.tsx`
5. **Modify love reasons** in `WhyILoveYou.tsx`

### Color Scheme:
Edit `tailwind.config.js` to customize colors:
- `romantic-pink` - Primary pink color
- `romantic-purple` - Primary purple color  
- `night-sky` - Dark background color
- `dream-blue` - Accent blue color

### Typography:
Fonts are loaded from Google Fonts:
- `Playfair Display` - Elegant serif for headings
- `Poppins` - Clean sans-serif for body text

## 🎵 Music Integration

The music player UI is ready for implementation. To add background music:

1. Add audio files to the `public/` directory
2. Implement audio controls in the HeroSection component
3. Use React hooks to manage playback state

## 📸 Adding Real Photos

To add your own photos to the Memory Gallery:

1. Replace the SVG placeholders in `MemoryGallery.tsx`
2. Update the `memories` array with real image paths
3. Add your photos to the `public/` directory

## 🔐 Secret Message Password

Default password: `2023-12-25` (December 25, 2023)

Change this in `SecretMessage.tsx` on line 11 to your actual anniversary date.
